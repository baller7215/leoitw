2a. If I utilize the term "all," what can I delete from the following code?

transition: background 0.3 ease, color 0.2s linear;

If you use the term "all" and want to have all the effects to have the same transitions, then you can delete "color 0.2s linear" and "background" thus replacing "background" with "all".


2b. Define the following timing functions/transitioning elements:

"CSS Transitions allow property changes in CSS values to occur sly over a specified duration."

  • Ease: a transition effect which begins slow, then speeds up, and ends slow
  • Linear: a transition effect with the same speed throughout the entire transition
  • Ease-In: a transition effect that begins slowly
  • Ease-Out: a transition effect that ends slowly
  • Ease-In-Out: a transition effect that begins and ends slowly
  • Cubic Bezier: a transition effect that allows you to use your own values in the Cubic-Bezier function
