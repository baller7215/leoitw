What makes a web design good? What are the four core elements of Design Focus?

Contrast

✅ Elements on a page should be strikingly different if not exactly the same
✅ Difference in Color, Fonts, Size


Repetition

✅ Create consistency. Consistency = Comfort.
✅ Think of shapes, line thickness, font selections, spatial relationships, size, texture


Alignment

✅ Create a hierarchy of importance for the content on your page on a separate document so everything is organized
✅ Be aware of the spacing between fonts, images, and web elements.


Proximity

✅ Group elements with similarities and distance elements with differences.
